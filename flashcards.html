Flashcards

Test your knowledge with these interactive flashcards.

What is Arduino UNO?

Microcontroller board based on ATmega328P.

What is a microcontroller?

A small computer on a single chip.

What is Arduino IDE?

Software to write and upload Arduino code.

Purpose of `pinMode()`?

Configures pin as INPUT or OUTPUT.

How to generate a tone?

Use `tone(pin, frequency);`

Use of `analogWrite()`?

Generates PWM signals.

Use of `Serial.begin()`?

Starts serial communication.

What is an interrupt?

Pauses code to handle an event.

Use of `map()` function?

Scales a value to another range.

What does `millis()` do?

Returns program uptime in ms.
