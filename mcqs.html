Multiple Choice Questions

Challenge yourself with these MCQs.

What is the default voltage of the Arduino UNO?

  • 3.3V
  • 5V
  • 12V

Correct Answer: 5V

Which microcontroller is used in Arduino UNO?

  • ATmega16
  • ATmega328P
  • ATmega2560

Correct Answer: ATmega328P

Which function is used to read an analog signal in Arduino?

  • digitalRead()
  • analogWrite()
  • analogRead()

Correct Answer: analogRead()

Which function is used to generate a tone in Arduino?

  • tone()
  • noTone()
  • playTone()

Correct Answer: tone()

Which function is used to attach an external interrupt?

  • attachInterrupt()
  • detachInterrupt()
  • enableInterrupt()

Correct Answer: attachInterrupt()

What does the `map()` function do in Arduino?

  • Converts analog input to digital
  • Maps a range of values to another range
  • Generates a sine wave

Correct Answer: Maps a range of values to another range
